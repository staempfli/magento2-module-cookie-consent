This Module is a wrapper for [Cookie Consent by Insites](https://cookieconsent.insites.com/)
 
## Installation  
  
Install the module with composer:  
  
```sh  
composer require staempfli/magento2-module-cookie-consent  
```  

  
## Requirements  
  
- PHP: 7.0.x | 7.1.x  
- Magento 2.1.x | 2.2.x
